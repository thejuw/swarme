# Competitor Pulse Skill

Discover where competitors rank and you don't, identify citation gaps on AI platforms.

## When to use
- User asks about competitor performance
- User wants to find keyword gaps
- Identifying where competitors are cited by AI but user isn't

## Data Sources
- Keywords table (gap_type field)
- Visibility checks against competitor URLs

## Output
- Keyword gaps with competitor URLs
- Search volume of missed opportunities
- Citation coverage comparison
